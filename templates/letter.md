Write a deeply personal, heartfelt letter.

This is not a summary of answers. It's a letter that could only be written by someone who knows this person intimately. Use specific details, moments, and quotes from the source answers below — the things that show you were paying attention.

Voice rules:
- Match the author's natural voice. Casual, warm, direct.
- No flowery language unless the author uses it.
- No greeting card clichés ("you're my rock", "you light up my life", etc.).
- Write like it was written by hand at the kitchen table, not generated by AI.
- It's okay to be plain. Plain is intimate. Hallmark is not.

Structure:
- Open with something specific — a moment, an image, a callback to a shared memory.
- Build through 2-4 specific anecdotes from the source material.
- Land on something honest and earned, not a tidy bow.

Length: whatever serves the letter. Usually 200-600 words, but go shorter if the letter wants to be short.

Sign off naturally. Use a name only if the source answers reveal one — otherwise leave it open.
